App Features & Screens

What the app contains

Live Dashboard (main screen)

  • Probe 1 — meat temp (large display)
  • Probe 2 — oven ambient temp
  • Target temp + progress bar (% to goal)
  • ETA countdown ("~14 min remaining")
  • Alert when target reached (sound + visual)
  • Connect / Disconnect button

Preset Picker

  • Beef: Rare 52° / Med-rare 57° / Medium 63° / Med-well 68° / Well-done 74°
  • Pork: Medium 63° / Well-done 71°
  • Chicken / Poultry: 74°
  • Lamb: Rare 55° / Med-rare 60° / Well-done 72°
  • Bread: Standard 93° / Sourdough 96°
  • Custom — type any target

History & Graph

  • Live temperature curve (both probes)
  • Target temp line overlaid
  • Session log — past cooks stored locally
  • Export session as CSV (optional)

ETA Algorithm

Rolling average of temperature rise rate over the last 5 minutes of readings → (target − current) ÷ rate = minutes remaining. Displayed once enough data is collected; shows "—" early in a cook when rate is unstable.

Data stored in localStorage

  • Active session: timestamped readings from both probes
  • Past sessions: start time, meat type, target, peak temp
  • Custom presets added by user
